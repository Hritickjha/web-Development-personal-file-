  • Project 3 : Game-flappy-bird--java (github)
    • The popular Flappy Bird game is available in Java in the "Game-flappy-bird--java-" repository, which was made by Hritickjha.   • Project 4 : Java_project (github)
    • Hritickjha has curated the "Java_project" repository, which is a useful teaching tool that highlights Java programming skills. There are three levels to its extensive course offering: basic, intermediate, and advanced. While basic programs cover basic subjects like Hello World, basic calculators, and string manipulation, intermediate programs study sorting and searching techniques like Bubble Sort and Binary Search. Advanced topics include object-oriented programming in-depth analyses, graph algorithms such as DFS and BFS, and dynamic programming techniques. Java solutions are simple to understand and implement for both developers and learners, since each application comes with extensive comments and explanations. The material in the repository, which showcases the collaboration and ongoing learning of the Java community, is open to improvements.

  • Project 5 : Full-Stack-App-python-Javascript(github)
    • The "Full-Stack-App-python-Javascript-" repository was made by Hritickjha as a comprehensive teaching project that demonstrates how to develop a full-stack application. The backend is designed in Python using modules like Flask for RESTful API creation, SQLAlchemy for database management, and Marshmallow for data serialization. On the front end, JavaScript is used with React and Redux to create interactive user interfaces and manage states. The frontend and backend portions of the repository each have their own directories containing the setup-related dependencies and configuration files (which are listed in requirements.txt and package.json, respectively). Users can clone the repository, install frontend and backend dependencies, create a virtual environment for the backend, and start the servers in order to run the application locally.
